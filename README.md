# Business Performance KPI Analysis (Power BI)

## Project Overview
This project focuses on analysing overall business performance using sales transaction data to track key performance indicators (KPIs) such as orders, revenue, quantity sold, and profitability.  
The goal was to provide a clear, executive-level view of business health and trends to support data-driven decision-making.

## Tools Used
- Power BI  
- Excel  
- Power Query Editor  
- Pivot Tables & Charts  

## Methodology
- Loaded and explored raw sales data  
- Cleaned and transformed the dataset using Power Query Editor  
- Created calculated fields and measures for core KPIs  
- Built pivot tables to validate metrics and trends  
- Designed interactive visuals and dashboards in Power BI  
- Analysed performance across time, regions, and product categories  

## Key Insights
- Total orders, sales, quantity, and profit were clearly tracked at a glance  
- Sales and profit showed consistent growth over time  
- Certain regions and product categories contributed significantly more to revenue and profit  
- Monthly sales trends highlighted peak and low performance periods  

## Outputs
- Interactive Business Performance KPI Dashboard (Power BI)  
- Cleaned and structured dataset  
- KPI cards, trend charts, and category/region breakdown visuals
